build

Build with options and body.

options

Add custom options

options(content: object): esb
Parameters
content (object) option body.
Returns
esb: Builder.
Example
esb().options({
  index: 'logs',
  type: '2016.01.01'
})
.build()

body

Add custom body

body(content: object): esb
Parameters
content (object) option body.
Returns
esb: Builder.
Example
esb().body({
   query: {
      match: {
         message: 'hello'
      }
   }   
})
.build()

indices

Add index field to option.

indices(indices: array, ignoreUnavailable: bool, allowNoIndices: bool, expandWildcards: string)
Parameters
indices (array) multi index
ignoreUnavailable (bool) Whether specified concrete indices should be ignored when unavailable (missing or closed)
allowNoIndices (bool) Whether to ignore if a wildcard indices expression resolves into no concrete indices. (This includes _all string or when no indices have been specified)
expandWildcards (string) Whether to expand wildcard expression to concrete indices that are open, closed or both.
Example
esb()
 .options()
 .indices(['2016.01.01'], true, true, 'open')
 .build()

type

type(args: ...any)
Parameters
args (...any)
Example

 .build()
